Intermediate Representations (IR) Editor

Soot has several different IR's, some of which can be displayed using this editor. It displays Jimple, Grimp, Baf and Shimple files.

This editor highlights keywords in the various IR's and works together with the IR Editor Content Outline View.

It also has the capability of displaying hover help and when they exist, displays tooltip hover help about Soot attributes and markers indicating which lines contain this information. Many more details are given in the Attributes section.
