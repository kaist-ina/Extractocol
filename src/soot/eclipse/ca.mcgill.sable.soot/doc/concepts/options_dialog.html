Soot Options Dialog

This dialog gives you a way to set any of the Soot options and then run Soot. There are approximately 180 options. This dialog stays in synch with the Soot options parsing code. Each time it is opened the defaults are reset. For saving specific configurations of options see Configurations Dialog.

Use the tree on the left to navigate to a section of options. Change specific options in the right panel. Each option has a tooltip explaining the option. Click Run to run Soot using the options you have selected. Click Close to close the dialog without running Soot.

Note:

At this time changing an option will not update other options that may be dependent on it. Hopefully this will be implemented in the future.
